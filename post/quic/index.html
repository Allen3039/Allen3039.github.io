网络模型

物理层：0101
数据链路层： 定义数据在局域网的传输
网络层:：IP协议 可以跨进行更大的互联网传播
传输层：
UDP 额外指明数据需要发送到哪个端口
TCP 除了UDP这些之外，做了额外的拥塞控制，快速重传
应用层: http https
